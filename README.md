# Standard Silverstripe template without pages
with Vite, SCSS and more, now with ddev config

## Installation
1. Clone this repo
2. Run ddev start
3. Run `ddev yarn`
4. Run `ddev composer install`
5. Run `ddev sake db:build`
6. Run `ddev yarn dev` to start the frontend dev server

## Build
Run `yarn build` to build the project. The build artifacts will be stored in the `app/dist/` directory.
